### Installation

1) DOCKER BUILD
Open CMD in `cccd-api-py` folder, run:

```python
docker-compose up --build
```

OR

2) 

```python
docker pull hiekichan/cccd_api
```

then:

```python
docker run -d -p 9096:9096 hiekichan/cccd_api
```
